## Inicio Rápido

Para comenzar a desarrollar:

1. Clonar el repositorio
2. Instalar dependencias: `npm install`
3. Configurar variables de entorno (ver `.env`)
4. Ejecutar servidor de desarrollo: `npm run dev`
